# 🫧 Abyssal Zen: Digital Bubble Popper

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Vibe: Deep Sea](https://img.shields.io/badge/Vibe-Deep%20Sea-075985)](#)

A high-fidelity, interactive "anti-stress" experience. Dive into the deep ocean where every click births a iridescent, 3D glassmorphic bubble. Watch them drift toward the surface or pop them to see them fragment into shimmering micro-bubbles.

---

## ✨ Features

* **🌊 Abyssal Background:** A dynamic, moving gradient simulating the deep sea.
* **💎 Glassmorphism 2.0:** Bubbles feature 3D depth, inner glows, and realistic light reflections.
* **🧘 Zen Mode:** Sit back and relax as bubbles autospawn and pop randomly.
* **💥 Fragment Physics:** Popping a bubble creates a burst of 5-6 micro-shards that fly outward.
* **🎨 Dynamic Hues:** Every bubble is uniquely tinted using random HSLA values.
* **🚫 Clean Interaction:** Zero text-selection flickering and purely linear vertical motion (no wobble!).

---

## 🚀 Quick Start

1.  Clone the repository:
    ```bash
    git clone https://github.com/AJ-016/digital-bubble-popper.git
    ```
2.  Open `index.html` in your favorite browser.
3.  **Profit.** (Relaxation is the currency here).

## 🕹️ Controls

| Action | Result |
| :--- | :--- |
| **Left Click** | Create a new bubble |
| **Click Bubble** | Pop it into fragments |
| **Zen Button** | Toggle automated tranquility |
